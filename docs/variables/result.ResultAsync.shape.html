      • Takes an object where values are ResultAsyncs or Promises of Results. Concurrently awaits all of them and returns a single ResultAsync containing an object with the identically key-mapped Ok values, or the first Err encountered.

        Note: When some values have never-typed parameters (e.g. bare chas.errAsync()), the inferred error union type may degrade to any. This is a known TypeScript inference limitation with complex recursive types. Results with concrete T and E types (e.g. from real functions) infer correctly.
